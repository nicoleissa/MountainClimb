# MountainClimb
> This Java program reads a text file with guidelines and shows a hiker's journey up a mountain. The main class is the MountainClimb class.
 
## Table of Contents
* [General Info](#general-information)
* [Setup](#setup)
* [Usage](#usage)
* [Project Status](#project-status)


## General Information
- This project implements a program that uses a binary search tree to model a mountain hiking expedition.  
- This project was assigned during a Data Structures course.

## Setup
You may use a text editor, such as Sublime, along with a command prompt on your device or an IDE, like VSCode, to run the program.

## Usage
MountainClimb.java file will be the file you want to run as this is the main class for the program. Hiker.java, BST.java, BSTMountain.java, and RestStop.java are accompanying classes that help the program run.
